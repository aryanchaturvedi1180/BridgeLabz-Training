## üéØ Topics Covered

### 1. Class and Object (`gcr-codebase/class-and-object/`)
Fundamental concepts of classes and objects in Java:
- **BankAccount**: Bank account management with deposit, withdraw, and balance display
- **Book**: Book information management
- **Circle**: Circle geometry calculations
- **Employee**: Employee data management
- **Student**: Student information system
- **CartItem**: Shopping cart item representation
- **MobilePhone**: Mobile phone attributes and methods
- **MovieTicket**: Movie ticket booking system
- **PalindromeChecker**: String palindrome validation

### 2. Inheritance (`gcr-codebase/inheritance/`)
Demonstrates inheritance, method overriding, and polymorphism:
- **AnimalHierarchy**: Animal class hierarchy with Dog, Cat, and Bird
- **BankAccountHierarchy**: Banking system with different account types
- **EmployeeManagementSystem**: Employee hierarchy and management
- **Library**: Library management with inheritance
- **MultilevelCourses**: Multi-level course inheritance
- **MultilevelOrders**: Order processing hierarchy
- **RestaurantHybrid**: Restaurant system with hybrid inheritance
- **SchoolRoles**: School role hierarchy (Student, Teacher, Principal)
- **SmartHomeDevices**: Smart home device inheritance
- **VehicleAndTransportSystem**: Vehicle and transport hierarchy
- **VehicleHybrid**: Hybrid vehicle inheritance

### 3. Java Constructors (`gcr-codebase/java-constructors/`)
Constructor implementations and access modifiers:
- **BankAccount**: Constructor with access modifiers (public, protected, private)
- **Book**: Book constructor with parameters
- **BookLibrary**: Library system with constructors
- **CarRental**: Car rental system
- **Circle**: Circle constructor implementation
- **Course**: Course management with constructors
- **Employee**: Employee constructor examples
- **HotelBooking**: Hotel booking system
- **LibraryBook**: Library book management
- **Person**: Person class with constructors
- **Product**: Product constructor examples
- **Student**: Student constructor implementations
- **Vehicle**: Vehicle constructor examples

### 4. Java Keywords (`gcr-codebase/java-keywords/`)
Usage of important Java keywords:
- **static**: Static variables and methods
- **final**: Final variables and classes
- **this**: Reference to current object
- **super**: Accessing parent class members
- **instanceof**: Type checking
- Examples in: Book, Employee, Patient, Product, Student, Vehicle

### 5. OOPs Pillars (`gcr-codebase/oops-pillars/`)
Complete implementation of the four pillars of Object-Oriented Programming:

#### Encapsulation
- Data hiding using access modifiers
- Getter and setter methods
- Private member variables

#### Inheritance
- Class hierarchies
- Method overriding
- Super keyword usage

#### Polymorphism
- Method overriding
- Runtime polymorphism
- Interface implementations

#### Abstraction
- Abstract classes
- Interfaces
- Abstract methods

**Implementations:**

## üöÄ Getting Started

### Prerequisites
- Java Development Kit (JDK) 8 or higher
- Any Java IDE (IntelliJ IDEA, Eclipse, VS Code) or command line

### Running the Code

1. **Compile Java files:**
   ```bash
   javac -d . path/to/YourClass.java
   ```

2. **Run Java programs:**
   ```bash
   java package.name.ClassName
   ```

3. **Example:**
   ```bash
   # Navigate to the directory
   cd gcr-codebase/class-and-object
   
   # Compile
   javac -d . BankAccount.java
   
   # Run
   java classandobject.BankAccount
   ```
